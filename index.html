Spritepacker Demo

SpritePacker packs many small images on to larger images and creates the corresponding css/json.

Download this project as a .zip file Download this project as a tar.gz file

Simple Usage

Create an instance from SpritePacker class and add your images.


$spriteDirectory = 'images/sprites';

$spritePacker = new SpritePacker();
$spritePacker->addFromDir($spriteDirectory);
$spritePacker->run();

With this, a 500x500px atlas will be generated under atlas/atlas.png with the corresponding css atlas/atlas.css.

Usage widh options



$options = array(
    'name' => 'options-atlas',
    'atlas-width' => 260,
    'atlas-height' => 400,
    'gutter' => 5,
    'render' => array(
        'RenderCSS',
        'RenderPNG',
        'RenderJSON',
    ),
);

$spriteDirectory = 'images/sprites';

$spritePacker = new SpritePacker($options);
$spritePacker->addFromDir($spriteDirectory);
$spritePacker->run();

With this, a 500x500px atlas will be generated under atlas/atlas.png with the corresponding css atlas/atlas.css.
